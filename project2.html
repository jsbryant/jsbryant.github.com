Sony PlayStation 2FA


Two Factor Authentication is an additional layer of security for the PlayStation user. A code is sent to the user's chosen mobile device which can then be used in tandem with their username and password to log them in to their account more securely.

As a Sr UI designer I worked closely with UX to create a quick, fun and digestible story that explained the importance of 2FA while simultaneously walking the user through the setup process.
